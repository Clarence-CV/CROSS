# CROSS Project Website

This is a static project page for:

**CROSS: Cascaded Distillation and Dual-Constraint Grounding for Remote Sensing Referring Segmentation**  
**ECCV 2026**

## Files

- `index.html` — main project page
- `style.css` — page styling

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder.
3. Make sure `index.html` is in the repository root, or configure GitHub Pages to serve this folder.
4. In GitHub repository settings, enable GitHub Pages.

## Things you may want to replace

- Update the `Paper`, `Code`, and `Dataset` links in `index.html`
- Add your institution names
- Replace the placeholder abstract with your final abstract
- Add method figures, qualitative examples, and benchmark tables
- Add a teaser image if desired
